# LedgerLink

LedgerLink is a modern financial management and dashboard application designed to provide organizations with actionable insights, seamless invoicing, payroll processing, and inventory management.

## Features

- **Dynamic Dashboards**: Sector-aware dashboards tailored for Schools, Churches, Law Firms, Clinics, and other organizations.
- **Invoicing & Payroll**: Manage your cash flow, track overdue payments, and handle staff disbursements easily.
- **Inventory Tracking**: Real-time inventory tracking with low-stock alerts.
- **Mobile Responsive**: Fully responsive layout with a slide-in mobile navigation drawer.
- **Real-time Data**: Powered by a unified Zustand store with localStorage persistence for seamless sessions.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the local link provided in the terminal (e.g., `http://localhost:3000`).
